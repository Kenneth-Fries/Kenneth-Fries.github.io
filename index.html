Covid 19

This is a quick dashboard I put together with data from John Hopkins University to keep track of Covid 19 around the country. I use a bash script on AWS to run daily a python program which aquires the data. It then processes the data into a JSON format, which is saved and synced via the bash script to github. There my html page makes use of the data by loading a javascript file that makes dynamic html content with dropdown selectors for user input on loaction of interest. Plotly is then utilized to display the default 7 day smoothed average data for the location's newly reported confirmed cases. Daily confirmed raw data amd daily reported deaths is also available to be ploted.
